\f0\fs24 \cf0 # Twitter Sentiment Analysis\
\
This project is a machine learning pipeline for analyzing public sentiment on Twitter data.\
\
## Overview\
\
The project focuses on collecting tweets, storing data in databases, and applying NLP methods to classify sentiment.\
\
## Features\
\
- Twitter API data collection\
- Python data processing scripts\
- SQL database workflow\
- NLP sentiment classification\
- Zero-shot learning approach\
- Data analysis and visualization\
\
## Technologies\
\
- Python\
- SQL\
- Jupyter Notebook\
- NLP\
- Machine Learning\
- Pandas\
- Matplotlib\
\
## Project Structure\
\
- `data_collection/` - scripts for collecting Twitter data\
- `database/` - database-related scripts\
- `modeling/` - notebook for machine learning modeling\
\
## Notes\
\
This project was developed as part of an undergraduate machine learning / NLP research project.}
